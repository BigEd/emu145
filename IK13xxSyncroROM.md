В эмуляторе как и в книге Трохименко синхропрограмма представлена последовательностью адресов микрокомманд в следующем виде -- A1, A2, A3, B1, B2, B3, C1, C2, C3


Строки -- ПЗУ разбито на 3 части по 32 строки. 32 строки адресуются через младшие 5 бит АСП. 3 части соответствуют индексу E в блоках A,B,C. (Верхняя часть E1, Средняя E2, Нижняя E3)
Столбцы -- ПЗУ разбито на 6 бит по 3 ниббла.  Биты в каждом ниббле стробируются декодированными старшими 2 битами АСП. 3 ниббла соответсвтуют блокам A,B,C - в следующем порядке --

```
~~~~bit 1~~~~~    ~~~bit 3~~~~~~    ~~~~ bit 5~~~~    ~~~bit 4~~~~~~    ~~~~bit 0~~~~~    ~~~bit 2~~~~~~
 B    A    C       B    A    C       B    A    C       B    A    C       B    A    C       B    A    C  
1032 1032 1032    1032 1032 1032    1032 1032 1032    1032 1032 1032    1032 1032 1032    1032 1032 1032
```

Строки в группе E1 нумеруются 0x00-0x1f сверху вниз. В группах E2 и E3 порядок строк обратный - 0x1f - 0x00 сверху вниз

ПЗУ инвертированно -- тоесть наличие перемычки интерпретируется как "0".
